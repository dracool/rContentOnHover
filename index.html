Reddit Content on Hover

Description

Reddit Content on Hover is a Greasemonkey/Tampermonkey script, that adds a preview of (some) images, videos or text posts to subreddits overviews.

Installing

If you do not have the Greasemonkey / Tampermonkey addon you can get them here:

Browser Addon
Chrome Tampermonkey
Firefox Greasemonkey Tampermonkey Beta
Opera Tampermonkey
Safari Tampermonkey

You can now install this script by clicking this link. This script has support for updates and will be updated occasionally with new functionality and fixes.

Features and Bugs

  • Reddit text posts
  • basic image types (png,jpg,gif)
  • gfycat.com
  • imgur.com single image links
  • twitter.com tweets

You can request sites to support or report bugs by creating an issue. Please do not request features that extend the scope of this project.
